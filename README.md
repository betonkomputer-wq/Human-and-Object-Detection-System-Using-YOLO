# Human-and-Object-Detection-System-Using-YOLO
Proyek ini membangun sistem pendeteksi manusia dan objek secara real time dari video dan kamera. Sistem mampu mendeteksi banyak objek dalam satu frame, menyesuaikan resolusi layar secara otomatis, dan menjaga performa inference tetap stabil. Cocok untuk kebutuhan monitoring, analisis visual, dan eksperimen computer vision.
